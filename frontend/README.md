# LuxeStore - Modern eCommerce Frontend

A premium, responsive, and feature-rich eCommerce frontend application built with React, Vite, Tailwind CSS, and Redux Toolkit.

## ✨ Features

- **Modern UI/UX**: Inspired by premium brands like Apple and Nike.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dark/Light Mode**: Smooth theme transitions with persistence.
- **Cart Management**: Add, remove, and update quantity using Redux Toolkit.
- **Product Filtering**: Advanced search and category-based filtering.
- **Animations**: Fluid page transitions and micro-interactions with Framer Motion.
- **Notifications**: Real-time toast notifications for user actions.
- **Dummy Data**: Pre-loaded with high-quality product data.

## 🚀 Tech Stack

- **Framework**: React.js (Vite)
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Notifications**: React Toastify

## 🛠️ Setup Instructions

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone or Download the Project**
2. **Navigate to the frontend directory**
   ```bash
   cd frontend
   ```
3. **Install Dependencies**
   ```bash
   npm install
   ```
4. **Run the Development Server**
   ```bash
   npm run dev
   ```
5. **Open your browser**
   Navigate to `http://localhost:5173`
